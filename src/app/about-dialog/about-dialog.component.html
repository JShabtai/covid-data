About

I made this tool so that I could interactively explore COVID data. Some of the goals were to identify milestones (such as peaks, decrease in case rates, etc...) and compare performance/progression in different countries.

Other useful tools

COVID-19 Dashboard by Johns Hopkins

An interactive map of confirmed cases, plus more that seems to be getting added frequently.

Covid Trends

A neat graph that plots number of daily cases against total cases, and uses animation to show time. Be sure to check out the video by Minute Physics linked at the top of the page.

COVID-19 Health Data

Compiles and displays lots of data including number of cases and healthcare resources and political orders (stay at home, school/business closures etc)... Also calculates projections which may be of interest.

COVID-19 Dashboards

A compilation of many different dashboards for COVID-19.

COVID-19 in Canada

A set of graphs, maps, and other visualizations specifically related to Canada.
