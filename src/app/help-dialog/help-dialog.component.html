Data Types

Confirmed

The total number of confirmed cases of COVID-19

Active

The number of active cases of COVID-19 each day (this is equal to confirmed - (recovered + deaths)

Recovered

The total number of recovered cases as of each day

Deaths

The total number of deaths as of each day

Graph Types

Daily

The number of cases of the selected type each day

Change

The daily change in the number of cases of the selected type, as a 7 day average

Example: Today there are 170 cases and 7 days ago there were 100. The change for today is (170 - 100) / 7 = 10.

Ratio

The percentage change as a 7 day average.

Other Options

Per 1000 people

The 'daily' and 'change' graphs will show the values per 1000 people in the country/region

Align to 100 cases

Start the graph for each region at the first day they had 100 cases
